RealmExample
============

This is a simple CRUD application for iOS app using Realm framework (http://realm.io/).

## Usage

Run:
	'pod install'

Open RealmExample.xcworkspace file to run de app.

## Model

Post:
- Title
- Text
- Publish Date
- Array of Comments


Comment:
- Comment
- Date

## Realm

- Insert
- Update
- Delete
- Select
